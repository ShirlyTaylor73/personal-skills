### #2 PaLM-E (Driess et al., ICML 2023)

- 标题：PaLM-E: An Embodied Multimodal Language Model
- 作者：Danny Driess, Fei Xia, Mehdi S. M. Sajjadi, et al.（Google / TU Berlin）
- 发表年份：2023
- venue：ICML 2023
- arXiv ID：2303.03378
- 中文摘要：本文提出"具身语言模型"PaLM-E，将连续的真实世界传感器模态（图像、状态估计、3D 神经场景表征 OSRT）通过编码器投影到 LLM 词嵌入空间，与文本 token 交织成"多模态句子"端到端送入预训练大语言模型 PaLM。模型解码生成自然语言决策（含 sub-goal 序列），由低层策略执行后再回到 PaLM-E 形成闭环。最大 PaLM-E-562B 由 540B PaLM 与 22B ViT 组成。在 TAMP、Language-Table、移动操作三个机器人域以及 OK-VQA、VQAv2、COCO 等通用视觉-语言任务上同时取得高水平表现，展示了从 web 数据到具身决策的正向迁移、多任务联合训练的数据效率，以及多模态思维链等涌现能力。
- 核心方法：通过"多模态句子" prefix 将图像/OSRT/状态向量经编码器 φ 注入预训练 PaLM 嵌入空间，与文本 token 同构地参与自回归生成；输出为自然语言形式的高层决策序列，经低层语言条件策略落地为机器人动作 〔p.4: "inject continuous, embodied observations such as images... into the language embedding space of a pre-trained language model"〕；引入 OSRT 3D 物体中心表征与 entity-labeling token 显著提升数据效率 〔p.5: "novel architectural ideas such as neural scene representations and entity-labeling multimodal tokens"〕
- 主要贡献：1) 提出可端到端训练、跨多个 embodiment 的具身多模态 LLM；2) 验证当前 SOTA VLM 零样本不能解决具身推理而 PaLM-E 可以；3) 引入神经场景表征 OSRT 与实体标签 token 等架构创新；4) 展示 PaLM-E 同时是有竞争力的视觉-语言通才（OK-VQA SOTA）；5) 证明语言模型规模放大可大幅减少多模态微调的灾难性遗忘 〔p.3: "(1) propose and demonstrate that a generalist, transfer-learned, multi-embodiment decision-making agent can be trained"〕
- 与本调研主题的关系：5.2A 大模型驱动 VLA 中的核心 VLM 基座论文：本组 #1 RT-2 直接以 PaLM-E-12B 为骨干构造 RT-2-PaLM-E 实例；PaLM-E 提出的"多模态句子 + LLM 嵌入注入"是后续闭源/企业系 VLA 通用范式之一。本身仍属"高层 planner + 独立低层策略"，尚未把动作 token 化（这一步由 RT-2 接力完成）。
- 优点：1) 多任务联合训练带来正向迁移，数据效率高（TAMP 仅 320 例可达高水平） 〔p.7: "we are mostly interested in the generalization aspects"〕；2) 对几何/物体姿态需要 fine-grained 推理的 TAMP 任务表现优于 SayCan、PaLI 零样本 〔p.8: "PaLM-E outperforms both PaLI and SayCan on embodied VQA and planning tasks"〕；3) 大规模 (562B) 微调后通用语言能力几乎不退化（仅 3.9%） 〔p.9: "merely 3.9% have been degraded for the largest model"〕
- 局限性：原文未集中给"limitations"专节，但讨论中点出冻结 LLM 路线在某些机器人任务上偶有挣扎，以及未来需要将 OSRT 与大规模视觉数据相结合等改进方向 〔p.10: "this approach occasionally struggled for robotics tasks"〕
- 典型应用场景：TAMP（任务和运动规划）多步抓放、Language-Table 桌面长程推动、移动机器人厨房长程取放与可供性预测/失败检测/重规划 〔p.6: "three robot environments (Fig.1) include a Task and Motion Planning (TAMP) domain"〕
- 数据集：TAMP 模拟（96k 训练场景，仅取 1% 即 320 例评估）、Language-Table（10/20/40/80 demos per task）、移动操作沿用 SayCan 的 2912 序列；通用视觉-语言用 WebLI、VQAv2、OK-VQA、COCO captioning 等 〔p.7: "96,000 training scenes of solely the TAMP environment"〕
- 评价指标：planning 与 embodied VQA 的成功率、F1（affordance/失败检测）、OK-VQA / VQAv2 准确率、COCO CIDEr 〔p.9: "failure detection and affordance prediction (F1 score)"〕
- benchmark 数值：
  - TAMP 1% data / planning 成功率(p1,p2 平均) / 94.9%（PaLM-E LLM finetune full mixture，12B） 〔p.7, Fig.4〕
  - Language-Table Task1 / 成功率 / 80.0%（PaLM-E-12B full mixture, 40 demos） 〔p.9, Tab.2〕
  - 移动操作失败检测 / F1 / 0.91（PaLM-E-12B full mixture LLM frozen） 〔p.9, Tab.4〕
  - OK-VQA val / 准确率 / 66.1（PaLM-E-562B 通才模型，单 checkpoint） 〔p.9, Tab.5〕
  - VQAv2 test-dev / 准确率 / 80.0（PaLM-E-562B 通才模型） 〔p.9, Tab.5〕
- 一句话评述：PaLM-E 是把 LLM 真正"接到"机器人传感器与具身规划上的标志性 VLM，是 RT-2 等首批 VLA 的直接父本，但其本身止步于高层 planner，不直接输出底层动作。
- 参考文献条目（GB/T 7714）：DRIESS D, XIA F, SAJJADI M S M, et al. PaLM-E: An embodied multimodal language model[C]//Proceedings of the 40th International Conference on Machine Learning. 2023.
